# One Place Store 🛒
Plataforma de E-commerce minimalista construida con React y Tailwind CSS.

## Características
- Diseño moderno y responsivo.
- Carrito de compras dinámico.
- Panel de administración para gestión de inventario.
- Estética Premium estilo Shopify.

## Cómo usar
Simplemente abre el archivo `index.html` en tu navegador o despliega en Netlify/GitHub Pages.
